# Clipord Setup Notes

## CRITICAL: Supabase Dashboard Configuration

### Fix OTP Email — MUST DO THIS

By default Supabase sends a magic link. You need to change it to send a 6-digit code.

1. Supabase Dashboard → **Authentication → Email Templates**
2. Select **"Magic Link"** template
3. Change the **Subject** to:
```
   Your Clipord code: {{ .Token }}
```
4. Change the **Body** to:
```html
   <h2>Your Clipord verification code</h2>
   <p>Enter this code in the app:</p>
   <h1 style="letter-spacing:8px;font-size:36px;">{{ .Token }}</h1>
   <p>This code expires in 10 minutes.</p>
   <p>If you didn't request this, ignore this email.</p>
```
5. Save template

6. Go to **Authentication → Settings**:
   - Set **OTP Expiry** to `600` (10 minutes)
   - Make sure **"Enable Email OTP"** is ON
   - **Disable "Confirm email"** (or users will need to click a link too)

### Fix Password Reset Email

1. Supabase → **Authentication → Email Templates → Reset Password**
2. The default template works. Just ensure the URL in the template uses `{{ .ConfirmationURL }}`
3. Go to **Authentication → URL Configuration**:
   - Site URL: `https://your-vercel-url.vercel.app`
   - Add to Redirect URLs: `https://your-vercel-url.vercel.app/reset-password`
4. Save

### PWA Install (Android Chrome)

For the install prompt to appear on Android Chrome:
1. App must be served over HTTPS ✅ (Vercel handles this)
2. manifest.json must have `display: "standalone"` ✅
3. Service worker must be registered ✅
4. Icons must exist at all specified paths ✅ (generated by build)
5. User must have visited the site at least twice with engagement

For iOS Safari: PWA install is always manual:
Share button (square with arrow) → Add to Home Screen

### Vercel Environment Variables

Set these in Vercel Dashboard → Project → Settings → Environment Variables:
```
VITE_SUPABASE_URL        = https://xxxxx.supabase.co
VITE_SUPABASE_ANON_KEY   = eyJ...
VITE_VAPID_PUBLIC_KEY    = BH... (from web-push generate-vapid-keys)
VITE_APP_NAME            = Clipord
VITE_APP_URL             = https://your-app.vercel.app
```

### Build Commands for Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist/pwa`
- Install command: `npm install`

### Extension Build (local)
```bash
npm run build:extension
```

Then load `dist/extension/` as unpacked extension in Chrome:
1. chrome://extensions → Enable Developer mode
2. Load unpacked → select dist/extension/

### Supabase SQL Schema

Run `supabase/schema.sql` in Supabase SQL Editor.
If tables already exist from a previous run, the `CREATE TABLE IF NOT EXISTS`
and `DROP POLICY IF EXISTS` statements handle re-runs safely.
